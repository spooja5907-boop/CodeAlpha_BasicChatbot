# 🤖 Basic Chatbot

## 📌 Description
This is a simple rule-based chatbot developed in Python as part of the CodeAlpha Python Programming Internship.

## ✨ Features
- Greets the user
- Responds to common questions
- Handles predefined inputs
- Continues chatting until the user types `bye`

## 🛠️ Technologies Used
- Python 3

## ▶️ How to Run
1. Run the Python program.
2. Type messages like:
   - hello
   - hi
   - how are you
   - what can you do
   - thank you
   - bye
3. The chatbot responds based on predefined rules.
